<a> 링크태그

a는 블록, 인라인이 모두 될 수 있다

a는 절대경로, 상대경로로 href 필수속성을 가진다.

a는 외부사이트(새창)를 띄울 경우만 target을 사용한다.

절대경로

파파고 바로가기

상대경로

절대적 웹 주소가 아닌, 현재 작업중인 파일을 기준으로
해당 폴더와 같은 폴더 또는 하위 폴더 또는 상위 폴더로 원하는 파일의 위치를 상대적으로 지정하는 경로

절대경로 웹주소 작성 시 시작 : http

상대경로 작성 시 시작 : ./ (현재 위치부터 시작하라는 명령)

상대경로 작성 순서

1. 현재 작업중인 파일의 위치를 확인한다.

2. 위 1을 기준으로 연결하고 싶은 파일의 위치를 확인한다.

3. ./로 경로를 시작하며 원하는 상대경로를 완성한다.

시작 파일 보기 (같은위치)
인형 찾기 (하위경로)
노트북찾기
휴지찾기
지우개찾기

상위경로

현재 작업파일 기준으로 연결하고자 하는 파일이 폴더 상위에 있는 경우

같은 위치 또는 하위 경로일 경우 시작 명령어 ./

상위 위치일 경우 시작 명령어 ../
